# bufferplot

Displays buffer log from liquidsoap prettily.

## Installation

	lein bin

Binary will be in target/bufferplot

## Usage

	./bufferplot "name for top of chart" /path/to/buffer.log output-filename.pdf


Output format is just pdf for now.
